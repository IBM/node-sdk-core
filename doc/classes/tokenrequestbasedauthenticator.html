Class TokenRequestBasedAuthenticator

Class for common functionality shared by token-request authenticators. TokenRequestBasedAuthenticators use token managers to retrieve, store, and refresh tokens. Not intended to be used as stand-alone authenticator, but as parent class to authenticators that have their own token manager implementations.

The tokens will be added as an Authorization headers in the form:

 Authorization: Bearer <bearer-token>
