Class Cp4dTokenManager

Token Manager of CloudPak for data.

The Token Manager performs basic auth with a username and password to acquire CP4D tokens.

Protected pacedRequestToken

  • pacedRequestToken(): Promise<any>

  • Inherited from TokenManager.pacedRequestToken

    • Defined in auth/token-managers/token-manager.ts:154

    Paces requests to request_token.

    This method pseudo-serializes requests for an access_token when the current token is undefined or expired. The first caller to this method records its requestTime and then issues the token request. Subsequent callers will check the requestTime to see if a request is active (has been issued within the past 60 seconds), and if so will queue their promise for the active requestor to resolve when that request completes.

    Returns Promise<any>
